## Slide 18: 项目执行心得 — 挑战与解决

| 挑战 | 解决方案 |
|------|----------|
| 四种不同页面架构如何统一 | 各自设计独立路由，主页通过 `Navigator.push` 分发 |
| SliverAppBar 中文字被 Tab 遮挡 | 使用 `Positioned(bottom: 100)` 为 Tab 栏预留空间 |
| 中韩翻译内容量大（500+行） | AI 辅助批量生成翻译数据 |
| 图片资源缺失 | AI 分析可用资源 + 补入 assets 路径映射 |
| 团队成员不同编码风格 | 建立统一的 models/ 数据模型和 widgets/ 公共组件 |
